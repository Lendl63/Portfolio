# Portfolio Template
Template de portfolio responsive avec Canvas animé

## Features
- Responsive design (320px-1920px+)
- Canvas galaxy background
- Lazy-loaded contact form
- WebP/AVIF images optimized
- SEO-ready (JSON-LD)
- Accessible (WCAG AA)

## Getting Started
1. Clone repo
2. Edit `index.html` - Replace placeholders:
   - [Your Name] → ton nom
   - [Your City] → ta ville
   - your.email@example.com → email
   - [+XX XXX XXX XXX] → téléphone
3. Run `npm install && npm run convert-images`
4. Deploy

## Tech Stack
- HTML5 / CSS3 / JavaScript (vanilla)
- Canvas API
- Sharp (image conversion)
- esbuild (optional minification)
